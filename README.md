# Evaluation Frontend

The **Evaluation Frontend** is a web application built with **Angular** for managing employee and external collaborator evaluations.  
It provides a clean and efficient interface for viewing assessments, filling out forms, and interacting with the evaluation system connected to a backend API [(https://github.com/rubensousa106/evaluation)]

---

## Features

- **Evaluation Management:** View and interact with evaluations for internal and external users.
- **Dynamic Forms:** Load and fill evaluation forms retrieved from the backend.
- **Dashboard Overview:** Quick access to available or pending evaluations.
- **API Integration:** Fully connected to a backend REST API for data exchange.
- **Component-Based Architecture:** Modular and reusable Angular components.

---

## Technologies Used

- **Angular** (TypeScript)
- **HTML5 / SCSS**
- **REST API** (Angular HttpClient)
- **Node.js & npm**
